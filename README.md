# 🐦 Chirpify

Welcome to **Chirpify** – a modern social media platform inspired by Twitter, built by **Younes El Hajri** and **Jayden Launuru**! 🚀

## 🌟 Features

✅ **User Authentication** – Sign up, log in.  
✅ **Post Messages** – Share thoughts with the world.  
✅ **Like & Repost** – Engage with content through likes and reposts.  
✅ **Comment System** – Reply to posts and reposts.  
✅ **Admin Dashboard** – Manage user content effectively.  
✅ **Responsive Design** – Works on all devices.  

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: PHP  
- **Database**: MySQL  

## 🎯 Goals & Future Updates

🚧 **Phase 1:** Basic functionalities (account registration, posting, liking).  
🔜 **Phase 2:** Image uploads, comment replies, improved UI.  
💡 **Phase 3:** Advanced features like notifications and DMs.
